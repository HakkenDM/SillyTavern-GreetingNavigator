# Greeting Navigator

A lightweight SillyTavern extension for browsing, searching and managing character greetings.

## Features

* 🔍 Search greetings instantly
* ⭐ Favorite greetings
* 🎲 Random greeting selection
* 🎨 Native SillyTavern theme integration
* 🔄 Live greeting switching
* 📝 Markdown rendering support
* ↔ Swipe counter synchronization
* ⌨ ESC to close

## Installation

1. Download or clone this repository.
2. Copy the folder into:

```text
SillyTavern/public/scripts/extensions/third-party/
```

3. Restart SillyTavern.
4. Enable the extension in the Extensions menu.

## Requirements

* SillyTavern 1.15.0+
* Release and Staging compatible
